# 🏥 Aarogyam Lifecare — Complete Website

Diagnostic & Pathology Centre | A unit of Motichand Hospital, Kushinagar

## Quick Setup

1. npm install
2. Edit .env (change ADMIN_PASS and EMAIL_PASS)
3. npm run dev  →  open http://localhost:3000

## Admin Panel
URL: /admin/login
Default: admin / Aarogyam@Admin2024  ← CHANGE THIS in .env

## Pages
/ → Homepage
/tests → Tests & Prices
/appointment → Book Appointment
/reports → Download Reports
/about → About Us
/contact → Contact

## IMPORTANT SECURITY
- Change ADMIN_PASS in .env before going live
- Change your GoDaddy email password (old one was exposed in git)
- Never commit .env to git
